# Tech-Lab Nexus
A modular collection of cybersecurity, AI, and web-based labs.

## Features
- **CyberLab**: SQL Injection challenge with AI feedback.
- **ResumeAnalyzer**: AI-powered resume scanner.
- **NewsAggregator**: Personalized tech news feed.
- **SoftSkillsTrainer**: AI-based chatbot for soft skills training.

## How to Run
1. Clone this repository.
2. Install dependencies from `requirements.txt` in each folder.
3. Run individual modules (`python app.py` or `docker run` for CyberLab).
